# Growth Marketing Skills

Agent skills for growth marketing workflows. Built for [Claude Code](https://claude.com/claude-code).

## Skills

### free-tool-research

Research competitor free tools, generate ideas, and build SEO-optimized free tool pages that drive organic traffic.

```bash
npx skills add vood/growth-marketing
```

## Usage

After installing, use in Claude Code:

- "research free tools" — analyze competitor tools and generate prioritized ideas
- "free tool ideas" — brainstorm tools for your product
- Build tool pages with JSON-LD schemas, FAQ sections, and conversion CTAs
